# Supported Languages

This is the list of programming languages parsed by
**singularity-code-analysis**, together with the current
state of metric coverage.

- Rust — full metric coverage
- Python — full metric coverage
- JavaScript — full metric coverage
- TypeScript — full metric coverage
- TSX (TypeScript + JSX) — full metric coverage
- Java — full metric coverage (and the only language with WMC today)
- C / C++ — full metric coverage aside from the macro parsing limitations noted in the README
- Kotlin — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- Go — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- C# — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- Elixir — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- Erlang — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- Gleam — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
- Lua — parsing and function detection are available; LOC, exit counting, and WMC metrics are not yet implemented
